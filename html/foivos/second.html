5G and Wifi 6

A trending technology in the past few years has been The Cloud.
The Cloud is actually the Internet. But when we refer to The Cloud, we mean storing or accessing data, to or from a remote location.
Remote servers are used to store data or run applications, so that these applications can be accessed from devices with much weaker hardware or storage capacity.
Applications that run in The Cloud are called Web Apps. A good example is Microsoft Office, which offers a cloud service.
